## 二分查找

```c
void Binaryserch(int*nums, int numsSize, int target){
    int left = 0;
    int right = numsSize - 1;
    while(left <= right){
        int mid = (right - left) >> 1 + left;
        if (nums[mid] == target){
            printf("找到了， 下标为%d\n", mid);
            return ;
        }
        else if(nums[mid] > target){
            right = mid - 1;
        }
        else{
            left = mid +1;
        }
        printf("没有要找的值\n");
    }
}
```
